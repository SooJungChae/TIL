## Sequence type

### list

- mutable 
```py
days = ["Mon", "Tue", "Wed", "Thur", "Fri"]
```

### Tuple

- immutable
```py
days = ("Mon", "Tue", "Wed", "Thur", "Fri")
```

## Dictionary

```py
soo = {
    "name": "Soo",
    "age": 22,
}
```

## Function

- `def` 로 함수 선언
- `:` 를 꼭 사용
- `{}` 없이 tab 으로 블록 처리

```py
def plus(a, b):
    print(a + b)

plus(2,3) // 5
```
